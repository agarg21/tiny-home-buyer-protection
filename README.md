# Tiny Homes Buyer Protection

Tiny Homes Buyer Protection is the internal project name for a U.S.-focused SEO/product experiment for practical tiny home buyer protection and planning tools. The public site brand is `Tiny Home Clarity`.

The project helps readers avoid expensive mistakes before they buy, build, finance, place, or live in a tiny home. The first priority is an interactive placement triage tool: `Where Can I Put a Tiny House?`

## Current Status

- Project stage: focused bootstrap experiment.
